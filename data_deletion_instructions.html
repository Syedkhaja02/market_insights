User Data Deletion

We comply with Facebook's requirements for data deletion callbacks.

API Endpoint

DELETE https://yourdomain.com/data-deletion/callback/

Request Body

{
  "user_id": "123456789",
  "app_id": "YOUR_APP_ID"
}

Response

On success, return HTTP 200 OK with an empty JSON object:

{}

Manual Deletion

If you prefer, you can email support@yourdomain.com with your account details and request deletion.
